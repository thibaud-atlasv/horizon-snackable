# Arena Vermin — Art Direction

Chibi isometric (dimetric) 2D pixel art style with oversized heads (1:1 head-to-body ratio), mandatory 2px black outlines, flat two-tone shading (base + 25% darker shadow), and no gradients on characters. Environment uses desaturated urban grays/greens while characters and pickups use fully saturated colors from a strict limited palette. Portrait canvas at 390×844 logical pixels with a fixed 45° horizontal / 30° vertical camera angle. All character animation is code-driven transforms on single static sprites — no spritesheet frame animation.
