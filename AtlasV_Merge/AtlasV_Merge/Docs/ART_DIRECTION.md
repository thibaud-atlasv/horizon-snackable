# ART_DIRECTION

**Naruto-themed, chibi face style with bold anime energy.** All merge items are iconic Naruto character faces only (no bodies) — round, expressive chibi heads with exaggerated cute expressions, recognizable at thumbnail size. The palette leans into Naruto's signature warm oranges, deep blues, and leafy greens, with each character's canonical color scheme preserved. The container evokes a giant ramen bowl or scroll, the background suggests the Hidden Leaf Village at golden hour, and UI elements use warm parchment tones with bold ninja-style typography.
