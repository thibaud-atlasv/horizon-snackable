# Greybox Validation Checklist

Run through before locking greybox for art handoff.

## Universal Checks

### Scale & Metrics
- [ ] Player capsule used consistently throughout
- [ ] All jumps tested against actual physics
- [ ] Cover heights match crouch/stand positions
- [ ] Doorways accommodate player + camera

### Flow & Pacing
- [ ] Critical path is clear without signage
- [ ] Pacing beats land as intended
- [ ] Backtracking feels purposeful (if present)
- [ ] Rest areas provide actual relief

### Playability
- [ ] Played by someone other than author
- [ ] Major issues addressed, not documented
- [ ] Edge cases tested (speedrun, exploration)

### Documentation
- [ ] Design intent documented per area
- [ ] Scale reference included for art
- [ ] Material/mood intentions noted
- [ ] Known issues logged

## Handoff Readiness
- [ ] Geometry named descriptively (not Box_047)
- [ ] Version control current
- [ ] Art team walkthrough scheduled
- [ ] Reference greybox preserved separately

## Red Flags to Address
- Player getting lost without visual landmarks
- Consistent confusion at specific points
- Pacing feels rushed or tedious
- Metrics don't match intended difficulty
