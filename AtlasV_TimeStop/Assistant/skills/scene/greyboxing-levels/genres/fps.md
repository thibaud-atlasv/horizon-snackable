# FPS Greyboxing

Priorities for first-person shooter level blockouts.

## Sightline Control
- Block views to prevent spawn-camping
- Create intentional engagement ranges: long / medium / short
- Test with scoped weapons at max range

## Cover Rhythm
- Space cover at sprint-time intervals (2-3 seconds between safe spots)
- Alternate hard cover (full block) and soft cover (partial)
- Ensure cover is contestable from multiple angles

## Map Control
- Identify power positions early; test if they're contestable
- Time rotations between key points
- Tune chokepoint widths for desired engagement density

## Verticality
- Height advantages must be contestable, not oppressive
- Provide multiple routes to elevated positions
- Test vertical combat with jump/climb mechanics

## Validation Method
Place weapons and spawns. Run 1v1s with another designer immediately. Record:
- Time-to-first-contact
- Average engagement range
- Power position dominance %

## Checklist
- [ ] Sightlines tested at all engagement ranges
- [ ] Cover spacing matches sprint timing
- [ ] Power positions have counterplay routes
- [ ] Rotation timings feel intentional
- [ ] Verticality is contestable
