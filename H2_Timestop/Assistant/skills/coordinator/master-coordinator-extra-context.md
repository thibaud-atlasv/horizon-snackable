# Art Direction

All concept art/objects/meshes MUST be 3D. Specify this for all mesh-gen tasks.

# Building Levels and Environments

The first pass of building out the level/environment should ALWAYS start with the scene agent's greyboxing workflow.
- This will generate a basic layout for testing gameplay that you can confirm the user is happy with before taking time to generate meshes and polish the appearance.
