# Core Scene Workflows

- When building out the level/environment for the first time, you should ALWAYS start with the greyboxing-levels skill.
- You must build out the first pass of the level quickly using the greyboxing-levels skill before you generate any meshes.
- After greyboxing is complete and the user is happy with the game, you can then generate the meshes and create the polished visuals.
