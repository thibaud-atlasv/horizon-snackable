# Racing Greyboxing

Priorities for racing game track blockouts.

## Racing Line
- Is the optimal path readable?
- Are there meaningful alternative lines?
- Do racing lines create passing opportunities?

## Corner Geometry
- Tune apex placement for satisfying turns
- Balance entry/exit widths
- Test banking angles for speed feel

## Hazard Timing
- Do obstacles create drama or frustration?
- Are hazards avoidable with skill?
- Test hazard placement at multiple skill levels

## Catch-Up Zones
- Where should rubber-banding opportunities exist?
- Are there strategic shortcut risks?
- Do straights allow for recovery?

## Lap Timing
Time laps obsessively. A 5-second difference in lap time is significant. Target:
- Consistent lap times for same skill level
- Variance from racing line choices, not confusion

## Validation Method
Run timed laps with multiple testers. Record:
- Lap time variance
- Corner-by-corner split times
- Collision/frustration points

## Checklist
- [ ] Racing line readable
- [ ] Corners tuned for feel
- [ ] Hazards dramatic, not frustrating
- [ ] Lap times consistent
- [ ] Passing opportunities exist
